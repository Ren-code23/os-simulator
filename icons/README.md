# Icons Folder

Place your PWA icons here:

- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels

## Quick Icon Generation

You can generate icons online at:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Or create simple icons using any image editor.

For now, the app will work without icons, but they're needed for proper PWA installation on mobile devices.
